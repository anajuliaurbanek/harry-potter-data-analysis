# Harry Potter Data Analysis

Projeto de análise de dados utilizando um dataset de personagens do universo Harry Potter.

## Tecnologias
- Python
- pandas
- matplotlib

## Dataset
Arquivo CSV com informações de personagens (casa, gênero, espécie, etc.).

## Análises realizadas
- Distribuição de personagens por casa
- Distribuição por gênero
- Espécies mais comuns

## Como executar
1. Instale as dependências:
   pip install -r requirements.txt
2. Execute:
   python analysis.py

## Objetivo
Praticar análise de dados, visualização e interpretação de resultados com Python.
